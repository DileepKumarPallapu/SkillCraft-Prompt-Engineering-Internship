# Task 3 - Prompting for Task Automation

## Objective

Use prompt engineering to automate repetitive data processing tasks.

### Prompt

Convert the following student details into JSON format.
Return only JSON.

Name: Dileep Kumar
Roll No: 22021A0501
Branch: CSE
CGPA: 8.5

### Output

```json
{
  "name": "Dileep Kumar",
  "roll_no": "22021A0501",
  "branch": "CSE",
  "cgpa": 8.5
}
```
